
# Chuck Norris App 

It's An Application containing Chuck Norris Jokes for Programmers, who doesn't like a good chuck norris joke!.

## Quick Overview

```sh
git clone git@github.com:Aydoun/chuck-norris.git
cd chuck-norris
npm start
```

Then open http://localhost:3001/ to see the app.<br>

### `npm test` 

Runs the test watcher in an interactive mode.<br>
By default, runs tests related to files changed since the last commit.

### `npm run lint` 

Runs eslint in fix mode.<br>
Extends recommended configuration for javaScript and React.
